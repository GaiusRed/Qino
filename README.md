# Qino

A Minecraft Modpack combining Cobblemon & Create on a Pre-Colonial Philippines-inspired Region.

## Modlist
An updated list of mods and their versions can be accessed here: https://modrinth.com/modpack/qino, along with modpack installation instructions.

### Featured Mods
- Cobblemon
- Create Fabric
- Terralith
- Incendium
- Nullscape

### Utilities
- BetterF3
- Comforts
- Continents
- Roughly Enough Items (REI)
- Universal Graves
- Waystones
- WTHIT
- Xaero's Minimap
- Xaero's World Map

### Optimization & Libraries
- Fabric Api
- Lithium
- Phosphor
- Sodium
- Indium
- Iris
- Architectury Api
- bad packets
- Balm
- Continuity
- Faithful 64x
- Solar Shader
- Your Options Shall Be Respected (YOSBR)
